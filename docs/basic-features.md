# 기본 기능 정리

작성일: 2026-05-06

이 문서는 AI 커플 다이어리 앱의 기본 기능만 정리한다. AI 고도화, 리포트, 스티커 확장, 테마 보상 등은 이후 단계에서 다룬다.

## 1. 기본 사용자 흐름

1. 사용자가 앱을 설치한다.
2. 소셜 로그인으로 가입한다.
3. 기본 프로필을 입력한다.
4. 커플 코드를 생성하거나 상대의 코드를 입력한다.
5. 커플 연결이 완료된다.
6. 오늘의 질문을 확인한다.
7. 각자 답변을 작성한다.
8. 두 사람 모두 답변하면 서로의 답변이 공개된다.
9. 답변은 커플 다이어리에 저장된다.
10. 다음 날 새로운 질문을 받는다.

## 2. P0 필수 기능

P0는 앱이 정상적으로 동작하기 위해 반드시 필요한 기능이다.

### 2.1 회원가입 및 로그인

- 사용자는 소셜 계정으로 가입 및 로그인할 수 있다.
- 우선 지원 후보는 Kakao, Apple, Google이다.
- 사용자는 로그아웃할 수 있다.
- 앱 재설치 또는 재로그인 시 기존 데이터를 불러올 수 있어야 한다.

### 2.2 사용자 기본 프로필

- 사용자는 닉네임을 입력할 수 있다.
- 사용자는 생일을 입력할 수 있다.
- 사용자는 연애 시작일을 입력할 수 있다.
- 사용자는 프로필 정보를 수정할 수 있다.

### 2.3 커플 연결

- 사용자는 본인의 커플 코드를 확인할 수 있다.
- 사용자는 상대방의 커플 코드를 입력해 연결 요청을 보낼 수 있다.
- 커플 연결은 1:1만 허용한다.
- 연결 완료 후 두 사용자는 같은 커플 공간을 공유한다.
- 사용자는 커플 연결을 해제할 수 있다.

### 2.4 홈 화면

- 홈 화면에는 오늘의 질문 상태를 표시한다.
- 홈 화면에는 커플 디데이를 표시한다.
- 홈 화면에는 상대방 답변 여부를 표시한다.
- 홈 화면에는 오늘 남긴 메모 또는 공지를 표시할 수 있다.

### 2.5 오늘의 질문

- 하루에 하나의 질문을 제공한다.
- 질문은 커플 두 사람에게 동일하게 제공된다.
- 질문은 날짜 기준으로 중복 제공되지 않아야 한다.
- 사용자는 오늘의 질문 상세 화면으로 이동할 수 있다.

### 2.6 답변 작성

- 사용자는 오늘의 질문에 답변을 작성할 수 있다.
- 답변은 텍스트 기반으로 작성한다.
- 답변 제출 전까지 수정할 수 있다.
- 답변 제출 후 수정 가능 여부는 정책으로 정한다.
- 사용자는 답변 작성 시 오늘의 감정을 선택할 수 있다.

### 2.7 답변 공개

- 한 명만 답변한 상태에서는 상대방의 답변을 볼 수 없다.
- 두 사람 모두 답변을 완료하면 서로의 답변을 공개한다.
- 답변 공개 시 상대방에게 푸시 알림을 보낼 수 있다.

### 2.8 답변 아카이브

- 완료된 질문과 답변은 날짜별로 저장된다.
- 사용자는 과거 질문과 답변을 다시 볼 수 있다.
- 사용자는 본인 답변과 상대 답변을 같은 화면에서 확인할 수 있다.

### 2.9 기본 알림

- 오늘의 질문 도착 알림을 보낸다.
- 상대방 답변 완료 알림을 보낸다.
- 미답변 리마인드 알림을 보낸다.
- 사용자는 알림 수신 여부를 설정할 수 있다.

### 2.10 기본 설정

- 사용자는 프로필 정보를 수정할 수 있다.
- 사용자는 알림 설정을 변경할 수 있다.
- 사용자는 커플 연결 정보를 확인할 수 있다.
- 사용자는 고객 문의 화면으로 이동할 수 있다.
- 사용자는 탈퇴할 수 있다.

## 3. P1 기본 경험 강화 기능

P1은 앱의 기본 사용성을 높이고 커플 앱다운 감성을 만드는 기능이다.

### 3.1 커플 메모

- 사용자는 상대에게 짧은 메모를 남길 수 있다.
- 메모는 홈 화면에 노출할 수 있다.
- 지난 메모는 보관함에서 확인할 수 있다.

### 3.2 답변 리액션

- 사용자는 상대 답변에 간단한 리액션을 남길 수 있다.
- 기본 리액션은 하트, 웃음, 감동, 고마워 정도로 시작한다.

### 3.3 답변 댓글

- 사용자는 공개된 답변에 댓글을 남길 수 있다.
- 댓글은 질문별로 묶어서 보여준다.
- 댓글 알림을 제공한다.

### 3.4 즐겨찾기

- 사용자는 특별한 답변을 즐겨찾기할 수 있다.
- 즐겨찾기한 답변은 별도 화면에서 모아볼 수 있다.

### 3.5 감정 기록

- 사용자가 선택한 오늘의 감정을 날짜별로 저장한다.
- 감정 기록은 월간 캘린더 형태로 볼 수 있다.

## 4. P2 이후 고려 기능

P2는 MVP 이후 사용성, 리텐션, 수익화를 위해 확장할 기능이다.

- 홈 화면 테마 선택
- 기본 스티커팩
- 사진 첨부
- 커플 위젯
- 월간 회고 카드
- 답변 배지
- 질문 분위기 선택
- 질문 재생성
- 프리미엄 구독

## 5. 기본 관리자 기능

- 사용자 조회
- 커플 연결 상태 조회
- 질문 등록 및 수정
- 오늘의 질문 배정 상태 확인
- 신고 및 문의 확인
- 앱 공지 등록
- 강제 업데이트 버전 관리

## 6. 기본 정책 결정 필요 항목

- 답변 제출 후 수정 가능 여부
- 커플 연결 해제 시 기존 답변 보관 여부
- 탈퇴 시 커플 데이터 삭제 기준
- 하루 질문 기준 시간
- 장거리 커플의 타임존 처리 방식
- 미성년자 가입 허용 범위
- 사진 첨부를 MVP에 포함할지 여부
- AI 기능을 기본 질문 생성부터 넣을지, 초기에는 운영 질문 은행과 함께 사용할지 여부

## 7. 기본 기능에서 제외

- 반려몽 또는 캐릭터 성장
- 캐릭터 방 꾸미기
- 복잡한 재화 시스템
- 공개 커뮤니티
- 다자 커플 또는 그룹 연결
- 전문 상담 기능
- 고급 AI 리포트
- 데이트 코스 추천
